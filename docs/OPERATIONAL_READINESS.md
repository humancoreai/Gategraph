# Operational Readiness Baseline

Release: v0.17.0_STABLE
Base: v0.16.9_STABLE
Status: candidate

This document defines a descriptive readiness surface for local single-node operation.
It does not introduce runtime authority, automatic repair, policy mutation, semantic scoring, deployment automation, or performance claims.

## Checked surfaces

- CLI entry point is declared
- server entry point is declared
- release-fast/profile surfaces are documented
- known operational boundaries are explicit
- generated local artifacts remain excluded

## Non-scope

- no hard throughput claim
- no enterprise deployment claim
- no autonomous recovery
- no automatic governance mutation
